Introducción

Este proyecto forma parte del curso de Minería de Datos y tiene como propósito aplicar técnicas de análisis descriptivo a un conjunto de datos climatológicos reales de Colombia. Mediante la implementación de un flujo de trabajo integral que incorpora herramientas modernas de análisis, procesamiento y visualización de datos, se busca extraer información valiosa y patrones significativos en los datos climatológicos del país.


El proyecto integra múltiples componentes tecnológicos: un modelo de datos dimensional (modelo estrella) que organiza la información de manera eficiente, una plataforma de visualización interactiva con Power BI, procesamiento distribuido con Apache Spark para análisis a escala, y herramientas de preparación de datos con Orange.


Además, se desarrolla una aplicación web en Flask como punto central de integración, permitiendo acceder a todos los componentes del proyecto de manera organizada y presentando los resultados de cada fase en un entorno unificado.

Justificación

Los datos climatológicos son fundamentales para comprender patrones ambientales, cambios en el clima y su impacto en sectores como agricultura, infraestructura y planificación ambiental. Colombia, como país con alta diversidad geográfica y climática, genera datos valiosos que requieren análisis profundos para extraer información útil.

  • Aplicación práctica de técnicas de minería de datos en un contexto real y relevante.
  • Integración de múltiples herramientas y tecnologías en un flujo de trabajo cohesivo.
  • Desarrollo de habilidades en análisis exploratorio, procesamiento distribuido y visualización de datos.
  • Generación de insumos que permitan entender patrones climatológicos a lo largo de diferentes períodos.

Marco Teórico: Minería de Datos

La minería de datos combina técnicas de estadística, inteligencia artificial, aprendizaje automático e ingeniería de bases de datos para explorar y analizar grandes volúmenes de datos con el objetivo de descubrir patrones, relaciones y conocimiento valioso. En este proyecto se enfatiza el análisis descriptivo como punto de partida.

Análisis Descriptivo

Fase fundamental que busca resumir y caracterizar los datos mediante estadísticas, visualizaciones y agregaciones. Responde preguntas como: ¿Cuáles son las precipitaciones promedio por región? ¿Existen anomalías o valores atípicos?

Modelo Dimensional

El modelo estrella organiza datos en tablas de hechos (medidas cuantificables como precipitación) y dimensiones (tiempo, ubicación geográfica, características de estaciones), facilitando análisis rápidos y eficientes.

Preparación de Datos

Etapa crítica que incluye identificación y tratamiento de valores faltantes, eliminación de duplicados, validación de rangos, y limpieza de datos malformados. La calidad de esta fase impacta directamente la confiabilidad de los análisis.

Visualización de Datos

Esencial para comunicar hallazgos de manera clara y accesible. Un dashboard interactivo permite explorar datos dinámicamente, seleccionar filtros y ver cambios en tiempo real, facilitando la toma de decisiones.

Metodología y Fases del Proyecto
F1
Modelo Estrella · Preparación y Modelado de Datos

Análisis exploratorio inicial del dataset, diseño del modelo estrella con dimensiones y hechos, y estructuración de datos para análisis multidimensional. Facilita consultas rápidas sobre datos climatológicos y la integración con herramientas BI.

F2
Dashboard Power BI · Visualización Interactiva

Conexión del modelo de datos con Power BI, diseño de dashboard con múltiples visualizaciones de distribución de precipitación por región y período, filtros interactivos por departamento, municipio, período y tipo de estación, e indicadores KPI.

F3
Apache Spark · Procesamiento Distribuido

Implementación de scripts en Apache Spark para procesamiento a escala. Cálculo de estadísticas descriptivas (media, mediana, desviación estándar) por dimensión, agregaciones por período temporal y región geográfica, e identificación de valores extremos y anomalías.

F4
Orange · Limpieza y Validación de Datos

Identificación y tratamiento de valores faltantes (NULL), eliminación de caracteres especiales, detección de duplicados y registros inconsistentes, validación de rangos y valores fuera de límites esperados, con visualización gráfica de resultados de limpieza.

Conocimiento Multidimensional

Esta sección presenta los resultados del análisis multidimensional sobre las Normales Climatológicas de Colombia a partir del modelo estrella implementado en SQL Server. Se realizaron consultas analíticas sobre las dimensiones de geografía, tiempo, período y parámetro climático, permitiendo identificar patrones de comportamiento climático a nivel departamental y por estación meteorológica.


El análisis se organiza en tres componentes: consultas analíticas fijas con visualizaciones de los principales parámetros climáticos, un explorador interactivo con filtros por parámetro, departamento y período, y una consola SQL libre que permite ejecutar consultas personalizadas directamente sobre el modelo estrella en la nube.
